## Props

| Property    | Type                                           | Default     | Description |
|-------------|------------------------------------------------|-------------|-------------|
| `variant`   | `"default" \| "destructive" \| "outline" \| "secondary" \| "ghost" \| "link"` | `"default"` | Visual style of the button. |
| `size`      | `"default" \| "sm" \| "lg" \| "icon"`          | `"default"` | Controls button height and padding. |
| `asChild`   | `boolean`                                      | `false`     | Replaces the button element with a `Slot` component. Useful for nesting links inside buttons. |
| `className` | `string`                                       | - | Additional CSS classes. |
| `...props`  | `ButtonHTMLAttributes<HTMLButtonElement>`      | â€”           | All native button attributes such as `onClick`, `disabled`, etc. |

## Variants

| Variant       | Description |
|---------------|-------------|
| `default`     | Solid button with primary background. |
| `destructive` | Red button for dangerous actions. |
| `outline`     | White background with border. |
| `secondary`   | Muted background. |
| `ghost`       | Transparent background with subtle hover. |
| `link`        | Looks like a text link. |

### Size

| Size      | Description |
|-----------|-------------|
| `default` | Standard button size (h-10, px-4). |
| `sm`      | Smaller compact version (h-9). |
| `lg`      | Larger button for emphasis (h-11). |
| `icon`    | Square icon button (h-10 w-10). |
