# CLI Installation

Install CapybaraUI into any project using our ShadCN-compatible CLI:

Step-1

```bash
npx shadcn@latest init
```
Step-2
```bash
npx shadcn@latest add https://uicapybara.vercel.app/r/basic-button.json
